【笔记】exKMP（扩展KMP）

发表于 2021-07-11 分类于 笔记 阅读次数：

概述

exKMP 主要用于在 O(n)O(n)O(n) 时间内解决求文本串 aaa 中所有后缀与模式串 bbb 的 LCP ，即最长公共前缀的长度。设 aaa 的长度为 nnn ，bbb 的长度为 mmm ，下标从 000 开始。设 p[i]p[i]p[i] 表示 a[i,n−1]a[i, n- 1]a[i,n−1] 与 bbb 的 LCP，题目即为求出所有的 p[i](i∈[0,n−1])p[i](i \in [0, n - 1])p[i](i∈[0,n−1]) 。不难发现，若存在若干个 p[i]=mp[i] = mp[i]=m ，则表示 bbb 在 aaa 中的 iii 位置完全出现，其实就是 KMP 要解决的问题，故该算法称为 exKMP。
